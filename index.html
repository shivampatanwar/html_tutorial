HTML Basics

HTML (HyperText Markup Language) is the standard markup language for creating web pages. It uses tags to structure content.

<!DOCTYPE html> - Declares the document type

Common Elements

  • <h1> - Heading level 1
  • <p> - Paragraph
  • <a> - Hyperlink
  • <img> - Image
  • <div> - Container

HTML Forms

Forms allow users to submit data. Key elements include:

  • <form> - Form container
  • <input> - Input field
  • <button> - Clickable button
  • <textarea> - Multi-line text

Semantic HTML

Semantic tags describe content meaning:

  • <header>, <footer>, <nav>
  • <article>, <section>
  • <main>, <aside>
