1. HTML Basics

HTML (HyperText Markup Language) is the foundation of all websites. It defines the structure, meaning, and organization of web content.

Key Points

  • HTML uses predefined tags.
  • HTML files are saved with .html extension.
  • Browsers interpret HTML and render pages.
  • HTML works with CSS (design) and JavaScript (logic).

Example

<!DOCTYPE html>
<html>
<body>
Hello World
</body>
</html>

2. HTML Syntax Rules

  • Tags must open and close properly.
  • Attributes must be inside opening tag.
  • Tags are nested correctly.
  • Lowercase is recommended.
<p class="text">Hello</p>

3. Document Structure

Every HTML document follows a hierarchical structure.

  • <html> → Root container.
  • <head> → SEO, metadata, styles.
  • <body> → Visible UI.
<html>
 <head><title>Page</title></head>
 <body>Content</body>
</html>

4. Text & Formatting

Text Tags

  • Headings control hierarchy.
  • Paragraphs separate content.
  • Formatting improves readability.
<h1>Main Title</h1>
<p>Description</p>
<b>Bold</b>

5. Hyperlinks

Links allow navigation between pages.

  • Absolute URL – full web address.
  • Relative URL – local file.
  • target="_blank" opens new tab.
<a href="page.html">Local Page</a>

6. Images

Images enhance visual communication.

  • Use optimized images.
  • Alt improves accessibility.
  • Width/height maintain layout.
<img src="logo.png" alt="Company Logo" width="120">

7. Lists

Used to display grouped data.

  • ul – unordered list.
  • ol – ordered list.
  • dl – description list.
<ol><li>Step 1</li></ol>

8. Tables

Tables represent structured data.

  • thead, tbody organize table.
  • Border collapse improves design.
<table>
<tr><th>Name</th></tr>
</table>

9. Forms

Forms enable interaction.

  • GET vs POST method.
  • Input validation.
  • Labels improve usability.
<input type="email" required>

10. Semantic HTML

Semantic tags provide meaning.

  • Improves SEO.
  • Improves accessibility.
<article>Blog Post</article>

11. Media

Embed multimedia content.

<audio controls></audio>

12. Meta Tags

Define browser behavior.

  • charset – encoding.
  • viewport – mobile scaling.
